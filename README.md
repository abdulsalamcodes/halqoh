# Halqoh Vibes

An Islamic audio lecture discovery platform built with Svelte.

## About

Halqoh is a directory of curated Islamic knowledge sessions (halqoh/sittings) compiled for easy discovery by Muslims worldwide. The goal is simple: help people find authentic lectures from trusted scholars.

We ask Allah to accept this effort as an act of Ibadah (worship).

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key_here
```
